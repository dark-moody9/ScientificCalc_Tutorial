# 🔥 Scientific Calculator (Java Swing)

A modern **Scientific Calculator** built using **Java Swing**, featuring a clean UI and support for both basic and advanced mathematical operations.

---

## 🚀 Features

- ➕ Basic Operations: `+  -  ×  ÷`
- 🧮 Scientific Functions:
  - `sin`, `cos`, `tan`
  - `log`, `ln`
  - `√` (Square Root)
  - Power `^`
- 🧠 Custom Expression Evaluation (No external libraries)
- 📜 Calculation History Panel
- ⚠️ Error Handling (e.g. Division by Zero)

---

## 🛠️ Technologies Used

- Java
- Java Swing (GUI)
- Stack Data Structure (Expression Evaluation)

---

## 📦 How to Run

1. Clone the repository:
```bash
git clone https://github.com/dark-moody9/scientific-calculator.git
```

2. Open in your IDE:
- NetBeans / IntelliJ / Eclipse

3. Run the file:
```bash


ScientificCalculator.java
```
